# Shop-locator
### An ASP.NET Core MVC Application
Languages:
- C#
- JavaScript
- HTML
- CSS

Tools and frameworks:
- Azure app service
- ASP.NET Core
- Vue.js
- Webpack
- SASS (scss)
- Visual Studio package - web compiler
- ADO.NET Entity Framework

### SQL Server required
A SQL server instance is required and therefore this application can only run on Windows.
